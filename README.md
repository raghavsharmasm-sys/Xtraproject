# 🌌 Ethara - Premium Team Task Manager

Ethara is a high-end, state-of-the-art SaaS platform designed for modern teams to collaborate, track projects, and manage tasks with a seamless, premium user experience.

![Dashboard Preview](https://via.placeholder.com/1200x600/0ea5e9/ffffff?text=Ethara+Dashboard+Preview)

## ✨ Features

- **💎 Premium UI/UX**: Built with Tailwind 4 and Framer Motion for a "living" interface with fluid animations and glassmorphism.
- **🌙 Global Dark Mode**: Seamlessly transition between Light and Dark themes with persistent state management.
- **📊 Interactive Kanban Board**: Manage tasks across multiple stages (Todo, In Progress, Completed) with a sleek drag-and-drop feel.
- **👥 Team Collaboration**: Add members to projects and assign tasks with ease.
- **📈 Real-time Activity**: Keep track of everything happening in your workspace with a dedicated activity feed.
- **🔒 Secure Auth**: JWT-based authentication for secure access to your data.

## 🚀 Tech Stack

### Frontend
- **React 19** & **Vite**
- **Tailwind CSS 4** (Design System)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Sonner** (Notifications)

### Backend
- **Node.js** & **Express**
- **MongoDB** with **Mongoose**
- **JWT** (Authentication)

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB (Local or Atlas)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/raghavsharmasm-sys/Xtraproject.git
   cd Xtraproject
   ```

2. **Backend Setup**
   ```bash
   cd backend
   npm install
   # Create a .env file and add your MONGO_URI and JWT_SECRET
   npm start
   ```

3. **Frontend Setup**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
